# TWS Gurkul Program Structure

## Current Programs:

### 1. **Crypto Mastery** (₹19,499)
- Foundation course for crypto trading
- 7-9PM Profit Window™ system
- For beginners to intermediate traders
- Single price point (no tiers)
- Path: Start here → Can upgrade to ETM later

### 2. **Footprint** ($497 / ₹34,997)
- Advanced institutional trading strategies
- W.H.A.L.E. Detection System™
- For experienced traders with larger capital
- Single price point (no tiers)
- Path: For advanced traders or after Crypto

### 3. **ETM (Elite Trader Mentorship)**
Three tiers available:

#### ETM LITE (₹6,999)
- 3 months of calls and tips only
- For budget-conscious traders
- No live sessions or mentorship
- Can start here directly

#### ETM PRO (₹14,999)
- 3 months with live session viewing
- Group calls and basic strategies
- Limited mentorship

#### ETM MAX (₹21,000)
- Full mentorship with live trading
- Personal WhatsApp group
- 1-on-1 sessions
- All bonuses included

### 4. **SuperStreams** (Separate product)
- Not part of the main trading education path

## Correct Learning Paths:

1. **Beginner Path**: 
   - ETM LITE (if just want tips) OR
   - Crypto Mastery → ETM MAX → Footprint

2. **Intermediate Path**:
   - Crypto Mastery → ETM PRO/MAX

3. **Advanced Path**:
   - Footprint → ETM MAX

4. **Budget Path**:
   - ETM LITE → Upgrade to Crypto when ready

## Key Points:
- NO "Crypto MAX" - Crypto is a single product
- NO "Footprint tiers" - Footprint is a single product
- ETM is the ONLY program with tiers (LITE/PRO/MAX)
- ETM MAX is available after Crypto OR Footprint
- ETM LITE can be purchased standalone
